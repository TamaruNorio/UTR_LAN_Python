## 7. トラブル時の確認項目

接続または応答に問題がある場合は、以下を確認します。

- [ ] 実機IPアドレスが正しい
- [ ] TCPポート番号が正しい
- [ ] LANモードがLANサーバーモードになっている
- [ ] PCと実機が同一ネットワークまたは到達可能なネットワークにある
- [ ] PC側ファイアウォールで通信が遮断されていない
- [ ] UTRRWManagerでは接続できる
- [ ] UTRRWManagerのTX/RXログとPythonサンプルのTX/RXが比較できる
- [ ] SUM不一致、ETX不一致、CR不一致が発生していない
- [ ] NACKの場合、エラーコードを `docs/protocol/` に基づいて確認している
